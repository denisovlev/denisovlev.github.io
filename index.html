Сортировка палитры

Embedding:

  • Original Palette = JPEG* → Indexed → Original Palette
  • Palette = Original Palette ⊗ Message

Detection:

  • Palette = Indexed → Palette
  • Original Palette = Indexed → JPEG* → Original Palette
  • Message = Original Palette ⊗ Palette

JPEG* в данном случае подразумевает не сжатие а только преобразование в "нормальный" вид

Embedding:

  • Original Palette = Indexed → Palette → Sort
  • Palette = Original Palette ⊗ Message

Detection:

  • Palette = Indexed → Palette
  • Original Palette = Palette → Sort
  • Message = Original Palette ⊗ Palette
